info О сервере

Этот сервер предоставляет безопасное HTTPS соединение для ваших веб-приложений. Поддерживает CGI скрипты, настройку CORS и песочницу chroot.

shield Безопасность

Сервер использует современные стандарты шифрования для защиты ваших данных:

  • SSL/TLS сертификаты
  • Изоляция процессов (chroot)
  • Запуск от непривилегированного пользователя
  • Контроль размеров запросов
  • Логирование событий

bolt Возможности

  • Многопоточная обработка запросов
  • Настройка CORS политик
  • Поддержка CGI скриптов
  • Гибкая конфигурация
  • Веб-интерфейс управления
  • Детальное логирование

api API Endpoints

Доступные эндпоинты сервера:

  • / — Главная страница
  • /settings — Панель настроек
  • /cgi-bin/ — CGI скрипты
  • /cgi-bin/config.cgi — API конфигурации

computer Технологический стек

Сервер построен на современных технологиях:

  • Backend: Чистый C (POSIX)
  • Оптимизация: GNU as (x86-32 ассемблер)
  • Компилятор: clang, as
  • Сетевой транспорт: mbedTLS 3.x
  • Маршрутизация: HTTP/CGI протоколы
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Параллелизм: POSIX потоки (pthreads)
